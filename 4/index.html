Pos Sifatnya terbarukan :
  1. Klik pos, kemudian tambah baru
  2. Tambahkan judul (terserah ketik apa)
  3. Jika mau menambahkan gambar, kemudian ambil dari Google, maka di rename menjadi JPG, kemudian di edit di paint baru di simpan
  4. Kemudian jika mau menambahkan gambar, pilih blog baru, klik gambar kemudian di unggah
  5. kemudian klik pos, tambahkan kategori baru (misalkan kebersamaan) kemudian di conteng dan terbitkan

Contoh selanjutnya :
