Cara membuat akun github
  1. Klik new repository
  2. Isi nama repository dengan nama anda
  3. Klik add a README file
  4. Creating repository
  5. Klik add file lalu pilih create new file
  6. Ketik angka 1/index.html
  7. Lalu isi dengan rangkuman dan klik commit new file.
  8. Kemudian pilih settings cari pages dan ganti none dengan main
  9. Dan klik save.
